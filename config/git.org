* Magit - A Git Porcelain
I use =magit= for git integration.
The following key-bindings are for global convenience.
#+BEGIN_SRC emacs-lisp
	(use-package magit
		:ensure t
		:bind (("C-x g" . magit-status)
				    ("C-x M-g" . magit-dispatch-popup)
				 )
		)
#+END_SRC
